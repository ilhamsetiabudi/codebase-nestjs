# Belajar Nest JS

code ini untuk siapapun yang ingin belajar nest js
## Installation

```
$ npm install
```
## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Run eslint

```bash
# unit tests
$ npm run lint:fix
```
